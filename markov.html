Astuces

Pour créer un sommet, cliquer dans une zone vide.

Pour déplacer ce sommet, maintenir [ctrl]+[clic]

Cliquer sur un sommet pour le selectionner. Appuyer sur [suppr] pour le supprimer.

Pour créer une arête, glisser avec la souris d'un sommet vers un autre.

Pour selectionner une arête, maintenir [ctrl]+clic.

Pour changer l'orientation d'une arête, la selectionner puis frapper [G]auche, [D]roite ou [A]ucun

Pour modifier le poids d'une arête, la sélectionner puis utiliser les boutons [P]lus ou [M]oins pour modifier son poids (ou son épaisseur).
